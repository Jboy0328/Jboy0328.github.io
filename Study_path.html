Final Challenge: Sales Data Analyzer

Problem: A company has sales data in an array of objects. Write a script that:

  • Calculates total revenue.
  • Finds the best-selling product.
  • Groups sales by category.
const sales = [
  { product: "Laptop", price: 999, quantity: 5, category: "Electronics" },
  { product: "Phone", price: 699, quantity: 10, category: "Electronics" },
  { product: "Desk", price: 250, quantity: 3, category: "Furniture" }
];

// 1. Total Revenue
const totalRevenue = sales.reduce((sum, item) => 
  sum + (item.price * item.quantity), 0);

// 2. Best-Selling Product (by quantity)
const bestSeller = sales.reduce((best, item) => 
  item.quantity > best.quantity ? item : best);

// 3. Group by Category
const salesByCategory = sales.reduce((groups, item) => {
  if (!groups[item.category]) groups[item.category] = [];
  groups[item.category].push(item);
  return groups;
}, {});

console.log({ totalRevenue, bestSeller, salesByCategory });
